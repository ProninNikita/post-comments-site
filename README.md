# post-comments-site

Сайт с постами и комментариями, Тетрис против ИИ, HTML-задания с проверкой.

## Файлы

- **index.html** — блог с постами и комментариями (localStorage)
- **tetris.html** — Тетрис против ИИ: золото, магазин, геймпад Xbox, звуки
- **html-exercises.html** — задания по HTML с автоматической проверкой

## Запуск

Открой любой .html файл в браузере.
